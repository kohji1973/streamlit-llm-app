# ホテルチェックインアプリ

ホテルのチェックイン用タブレットアプリケーションです。

## 機能

1. **4言語対応**（現在日本語のみ動作）
   - 日本語 🇯🇵
   - 英語 🇺🇸（今後対応予定）
   - 韓国語 🇰🇷（今後対応予定）
   - 中国語 🇨🇳（今後対応予定）

2. **予約検索**
   - お名前
   - 予約番号
   - 電話番号

3. **予約確認**
   - 予約内容の確認メッセージ表示

4. **お客様情報登録**
   - 国籍
   - 住所
   - パスポート情報（外国人客の場合）

5. **同伴者登録**
   - 2名以上の場合、同伴者の情報を登録

6. **宿泊税計算・徴収**
   - 料金に応じた宿泊税の自動計算

7. **QRコード表示**（日本人客のみ）
   - スタンプラリーポイント加算用

8. **部屋番号表示**
   - チェックイン完了後、部屋番号を表示

## セットアップ

### 1. 依存関係のインストール

```bash
pip install -r requirements.txt
```

### 2. 背景画像の設定

受付スタッフの画像を `images/reception_staff.png` として保存してください。

画像が配置されていない場合、デフォルトのグラデーション背景が使用されます。

### 3. アプリの起動

```bash
streamlit run hotel_checkin_app.py
```

## 使い方

1. **言語選択**
   - 最初の画面で言語を選択（現在は日本語のみ動作）

2. **予約検索**
   - お名前、予約番号、または電話番号で検索
   - サンプルデータ：
     - 名前: 山田太郎、予約番号: RSV001
     - 名前: 田中花子、予約番号: RSV002
     - 名前: John Smith、予約番号: RSV003

3. **予約確認**
   - 予約内容を確認して「はい」を選択

4. **お客様情報登録**
   - 国籍、住所を入力
   - 外国人客の場合はパスポート番号も入力

5. **同伴者登録**（該当する場合）
   - 同伴者の名前を入力
   - 外国人客の場合はパスポート番号も入力

6. **宿泊税支払い**
   - 計算された宿泊税を確認して支払い完了

7. **QRコード表示**（日本人客のみ）
   - スタンプラリーポイント加算用QRコードを表示

8. **部屋番号表示**
   - チェックイン完了
   - 30秒後に自動的に最初の画面に戻るか、「最初に戻る」ボタンで戻る
